Add `cast_value` and `scale_offset` v3 codecs

Implements the [`cast_value`](https://github.com/zarr-developers/zarr-extensions/tree/main/codecs/cast_value)
and [`scale_offset`](https://github.com/zarr-developers/zarr-extensions/tree/main/codecs/scale_offset)
codecs from zarr-extensions, enabling reads of arrays that use these
array-to-array transformations.
